# boa

Reactive stylesheet JS library (and eventually language).

## Usage

Basic usage is as follows:

    Boa.source(selector, property).bindTo(selector2, property2);

For advanced usage, see the documentation (when I write it).

## Tests

    gulp test

or

    gulp test-watch

## Contributing
